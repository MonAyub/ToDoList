
# 📋 To-Do List (C++)

A simple console-based task management application built with C++.

## Features
- ✅ Add new tasks
- 📄 View all tasks
- 🗑️ Delete tasks
- 🚀 Simple and clean interface

## How to Run
1. Compile: `g++ main.cpp -o todo`
2. Run: `./todo`

## Technologies Used
- C++
- Standard Library
